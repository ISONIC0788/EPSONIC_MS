# EPSONIC_MS

This project allows users to sign in, sign up, upload, view uploaded photos, perform CRUD operations, create a database, and view it.

## Features

- User Authentication
  - Sign In
  - Sign Up
- Photo Management
  - Upload Photos
  - View Uploaded Photos
- CRUD Operations
- Database Management
  - Create Database
  - View Database
